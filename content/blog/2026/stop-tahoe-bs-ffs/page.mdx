## 关闭上下文菜单中不明所以的图标

<MDXImage src="./resources/context-menu.png" />

Tahoe 的右键菜单里多了一些莫名其妙的图标，看着碍眼。一行命令关掉：

```bash
defaults write -g NSMenuEnableActionImages -bool NO
```

## 增大窗口边缘可拖拽区域

macOS 窗口边缘的拖拽区域一直很窄，经常对不准。可以通过这个命令增大外侧的可点击区域：

```bash
defaults write -g AppleEdgeResizeExteriorSize 8
```
