## 介绍


<MDXImage src="./resources/4e09bbb9.png" alt="" />


插件本身支持直接调用 DeepL API，但是因为 DeepL 的 API 计划收费较高让很多人望而却步。为此，我又额外开发了一个 API 可以让你以比较低的价格使用 DeepL API。如果感兴趣可以到 [这里](https://a-translator.royli.dev/) 查看。对了，这个服务叫 A Translator。


我已经开发了支援 A Translator 的 Shortcuts, Taio 和 Bob 的插件，可以让你非常爽地在 iOS 和 macOS 上使用 DeepL 服务。你可以在后台找到他们的下载地址。
